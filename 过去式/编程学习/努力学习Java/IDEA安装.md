### 快捷键

`Ctrl + Y`：删除当前行

`Alt + /`：补全代码

`Ctrl + /`：注释/取消注释

`Ctrl + 1 + Ctrl  + F`：代码格式化，自定义，因为被占了。

`Alt + Insert`：生成构造器等

`Ctrl + H`：查看一个类的层级关系

`Ctrl + B`：将光标放在一个方法上，可以选择定位到哪个类的方法

后面加.var：自动分配变量名
