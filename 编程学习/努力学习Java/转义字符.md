Java中常用的转义字符

1. `\t`：一个制表位，实现对齐的功能
2. `\n`：换行符
3. `\\`：yige `\``
4. ``\"`:		一个`"`
5. `\'`：一个`'`
6. `\r`：一个回车。



**注意**

换行和回车不一样

```java
        System.out.println("Hello\rWorld"); 
        // World
```

```java
        System.out.println("Hello\r\nWorld");
		// Hello
		// World
```
